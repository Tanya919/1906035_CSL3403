Styles in HTML

CASCADING STYLE SHEETS

Cascading Style Sheets(CSS) is a markup language responsible for how our web pages will look like.It controls the colors, fonts and layouts of our website elements. Without CSS, our website will appear as a plain HTML page.

HTML Selector

All HTML tags have default properties. With CSS we can define any HTML tag to change the defaults.
p { line-height: 14px; }

CLASS

A class is a rule that can be applied to an HTML tag. We can give a class any name we want with the exception of a few reserved words
.myclass { font: bold 14px Verdana; }

ID

ID rules are similar to a class, but they can only be applied once on a page to a particular HTML tag.
#myclass{ font: bold 14px Verdana;}
