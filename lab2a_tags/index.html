Difference between Tags & Attributes

HTML Tags HTML Attributes
  1. Tags contain elements which define how the information on a webpage is displayed.
  2. Tags do not appear on webpage.
  3. Tags are not case-sensitive.
  1. Attributes is a property that provides additional information about an HTML element.
  2. Attributes is specified inside opening tag.
  3. Attributes consists of two parts-name and value.
